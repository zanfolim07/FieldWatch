# FieldWatch

Projeto desenvolvido para a Global Solution 2026/1 da FIAP.

## Descrição

A FieldWatch é uma plataforma de monitoramento agrícola inteligente que utiliza imagens de satélite, Inteligência Artificial, sensores e chatbot para auxiliar produtores rurais na identificação de riscos como seca, pragas, queimadas e necessidade de irrigação.

A proposta é transformar dados agrícolas em informações simples e úteis para apoiar a tomada de decisão no campo, tornando o monitoramento da lavoura mais eficiente, acessível e sustentável.

## Objetivo

O objetivo do projeto é oferecer uma solução tecnológica para produtores rurais acompanharem suas lavouras em tempo real, recebendo alertas, recomendações e informações importantes para reduzir perdas, economizar recursos naturais e melhorar a produtividade.

## Funcionalidades

* Monitoramento da saúde da lavoura;
* Visualização de indicadores como NDVI e NDWI;
* Mapa da fazenda dividido por talhões;
* Identificação de áreas em situação saudável, atenção ou risco;
* Recomendação de irrigação inteligente;
* Consulta de alertas agrícolas;
* Assistente inteligente FieldBot;
* Formulário de contato com validação em JavaScript;
* Layout responsivo para diferentes tamanhos de tela.
